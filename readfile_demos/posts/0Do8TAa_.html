如果没有多线程，大概也不会去想各种东西。

因为程序基本上就是像你看到的那样一条一条顺序往下执行的。（we know it when we see it ）

在各种机器上跑相同的 Java 程序基本上都会有确定的行为 ，这一点是分析程序的基础。

 

但有了多线程之后 ，程序的行为仿佛多了 一层神秘的色彩。

为了能够分析清楚更复杂的多线程程序，补充一些单线程相关的知识是必要的。
