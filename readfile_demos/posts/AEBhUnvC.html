1、找放项目的文件夹

找到一个放项目的文件夹，在该文件夹路径下打开CMD

2、创建一个项目原型

执行下面的命令：

mvn archetype:generate -DgroupId=com.mycompany.app -DartifactId=my-app -DarchetypeArtifactId=maven-archetype-quickstart -DarchetypeVersion=1.4 -DinteractiveMode=false

在CMD窗口看到BUILD SUCCESS时，说明项目已经创建好了，这样就有了一个Maven风格的项目。

关于字段的含义（来自HOW2J）：

  • archetype:generate 表示创建个项目
  • -DgroupId 项目包名: com.mycompany.app
  • -DartifactId 项目名称: my-app
  • -DarchetypeArtifactId 项目类型: maven-archetype-quickstart
  • -DinteractiveMode:false 表示前面参数都给了，就不用一个一个地输入了

3、编译运行该项目

创建好Maven项目后我们需要将CMD当前路径切到项目的根目录（有pom.xml的目录下）。

pom.xml文件是Maven中项目配置的核心。它是一个单独的配置文件，包含了构建项目所需的大部分信息。

在项目根目录下执行下面的命令：

mvn package

这就类似于执行了一个自动帮我们编译，跑测试，打包的脚本。生成的东西放在target目录下：

 target中的内容

运行编译好的程序：

java -cp target/my-app-1.0-SNAPSHOT.jar com.mycompany.app.App
