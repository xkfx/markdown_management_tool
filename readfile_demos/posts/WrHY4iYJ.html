static关键字

1、static在作用于某个字段（也叫：类的数据成员、实例域）的时候，必然会改变数据的创建方式（创建的时机、位置）。

 

Java程序

// HelloData.java
import java.util.*;

public class HelloData {
    public static void main(String[] args) {
        System.out.println("Hello , it's: ");
        System.out.println(new Date());
    }
}

1、Sun公司提供的文档

 

2、java.lang：被自动导入到每一个Java文件

 

3、System.out：System类中的一个静态 PrintStream对象。

1）因为是静态的，不需要创建就已经存在。

2）因为System是java.lang里的类，java.lang被自动导入到每一个Java文件。

所以System.out.println()可以直接使用：它的作用是”将我给你的数据打印到控制台后换行“。

 

4、System.out.println(new Data());。（相关知识：toString()、垃圾回收机）
