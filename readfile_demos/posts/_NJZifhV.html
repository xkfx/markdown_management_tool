要求编写一个简单的人员信息管理程序，具体要求如下：

（1）程序涉及到五个类，分别是employee，technician，salesman，manager，salsemanager。这五个类的关系为：employee是顶层父类，technician，salesman，manager是employee的子类，salsemanager是salesman，manager的子类。

（2）employee具有员工号，姓名，收入等属性，有设置姓名，获取姓名，获取员工号，获取收入等函数。

（3）technician的收入计算方法为工作小时数*时薪，另外具有设置工作小时数，时薪的函数，具有工作小时数和时薪的属性。salesman的收入计算方法为销售额*提成比例，另外具有设置销售额，提成比例的函数，具有销售额和提成比例的属性。manager的收入计算方法为固定月薪，另外具有设置固定月薪的函数，具有固定月薪的属性。salsemanager收入计算方法为销售额*提成比例+固定月薪。

（4）要求利用employee类型的指针数组，指向8个子类对象（technician，salesman，manager，salsemanager每个类2个），通过相应方法输出这些对象的姓名（姓名的初始化通过键盘录入），员工号，收入（多态的形式）。

 

#include <iostream>
#include <cstdio>
#include <string>
using namespace std;

// 1-employee
class Employee {
public:
    // constructor
    Employee(int nu, string na, double sa): number(nu), name(na), salary(sa) {}

    virtual void display() const {
        printf("employee: %d %s salary=%.2lf\n", number, name.c_str(), salary);
    }

protected:
    // info of Employe
    int number;
    string name;
    double salary;
};

// 2-technician
class Technician : public Employee {
public:
    // constructor
    Technician(int nu, string na, int h, double money): Employee(nu, na, h*money) {}

    void display() const {
        printf("technician: %d %s salary=%.2lf\n", number, name.c_str(), salary);
    }
};

// 2-salesman
class Salesman : virtual public Employee {
public:
    // constructors
    Salesman(int nu, string na, double sa): Employee(nu, na, sa) {}
    Salesman(int nu, string na, double percent, double money): Employee(nu, na, percent*money) {}

    virtual void display() const {
        printf("salesman: %d %s salary=%.2lf\n", number, name.c_str(), salary);
    }
};

// 2-manager
class Manager : virtual public Employee {
public:
    // constructors
    Manager(int nu, string na, double sa): Employee(nu, na, sa) {}

    virtual void display() const {
        printf("manager: %d %s salary=%.2lf\n", number, name.c_str(), salary);
    }
};

// 3-salsemanager
class Salsemanager : public Manager, public Salesman {
public:
    Salsemanager(int nu, string na, double percent, double money, double Money):
        Employee(nu, na, percent*money+Money), Manager(nu, na, percent*money+Money), Salesman(nu, na, percent*money+Money) {}

    virtual void display() const {
        printf("salsemanager: %d %s salary=%.2lf\n", number, name.c_str(), salary);
    }
};

void showInfo(Employee& x) {
    x.display();
}

// unit test
int main()
{
    const int N = 5;
    Employee* p[N];

    Employee em(12301, "wang", 3200);
    p[0] = &em;

    Technician te(12302, "liu", 30, 100);
    p[1] = &te;

    Salesman sa(12304, "li", 0.7, 4000);
    p[2] = &sa;

    Manager ma(12305, "zhu", 4000);
    p[3] = &ma;

    Salsemanager sal(12307, "yao", 0.7, 1000, 7000);
    p[4] = &sal;

    for (int i = 0; i != N; ++i) {
        showInfo(*p[i]);
    }

    return 0;
}

 

1.动态绑定的出现场景通常是函数参数（基础类引用）作为子类对象的持有者。
2.只需把基类方法声明virtual，其子类对它的覆盖默认为虚函数。
3.如果不把基类方法声明成virtual而在子类对它进行覆盖，其父类引用仍可以持有子类对象，但是当通过父类引用调用方法的时候，将调用父类的方法而不是子类的方法（不会进行动态绑定）。
