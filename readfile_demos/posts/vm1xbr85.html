笔记:

1、确保导入mysql-connector-java-xxxx-bin.jar

2、sql语句容易出错，例如在组合String的时候漏掉空格

3、设置jlist的方法有很多种，具体参考api

4、继承一个类并不意味着继承父类的构造器， 原因是父类的私有域只能通过调用父类的公开方法来初始化。

5、要正确显示TableMode提供的列名，必须为table加上scrollPane。

        TableModel model = new DbTableModel(dbName , tableName);
        JTable table = new JTable(model);
        JScrollPane sp = new JScrollPane(table);

        add(sp);
        pack();

 6、鼠标右键对应的事件是e.getButton() == 3，如果不知道具体事件是什么，可以采用system.out.println(event)

7、对象间传递对象（共享同一个对象），可以通过写一个 accept方法，例如 a.accept(b.table)
