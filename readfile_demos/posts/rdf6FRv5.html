MySQL 5.5卸载

①关闭程序

②卸载程序。直接用360软件管家卸载，然后在C盘删掉整个MySQL文件夹

③编辑环境变量。将...MySQL Server 5.5\bin从环境变量Path中删除

MySQL 5.7安装配置

①MySQL分社区版和企业版，找到社区版进行下载，因本人操作系统是windows，所以先下载相应版本的MySQL Installer for Windows

因为网络问题，会有亿点点慢，建议下载之前先“优化”网络。

②开始安装步骤。因为主要是服务于Java开发，选择Server only就行了，客户端程序没什么用，MySQL Connectors也可以在开发时通过maven很方便的添加

③配置用户和密码。其它全部保持默认配置Next就行。

④配置环境变量。方便起见，把mysql文件下的bin目录添加到系统变量Path中去。

⑤测试。打开cmd，输入mysql -u用户名 -p敲回车输入密码连接mysql，连接成功则安装配置成功。
