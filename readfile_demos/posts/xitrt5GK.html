问题描述

操作环境win10，pthon以及pip版本如下

D:>py --version
Python 3.8.0
D:>py -m pip --version
pip 22.0.3 from ... (python 3.8)

升级pip之后（变成了如上版本）尝试pip install notebook以及pip install pillow都报如下错误

ERROR: Could not find a version that satisfies the requirement <包名> (from versions: none)
ERROR: No matching distribution found for <包名>

解决办法

参考：https://zhuanlan.zhihu.com/p/361790784

pip install 包名 -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
