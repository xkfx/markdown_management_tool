1、创建异常类。

步骤：1）定义一个派生于（继承）Exception或者其子类的类。

2）添加两个构造方法：一个是默认构造器，另一个是带有详细描述信息的构造器。

import java.io.IOException;

/**
 * Created by xkfx on 2017/2/16.
 */
public class MyException extends IOException{
    public MyException() {}
    public MyException(String gripe) {
        super(gripe);
    }
}

补充：继承中的构造方法（不写默认的有个隐式的构造方法）

 

2、捕获异常。

要想捕获一个异常，必须设置try/catch语句块。示例如下：

try {
     // code
     // more code                         
} catch (ExceptionType e) {
     // handler for this Exception
}

如果在try语句块的任何代码抛出了一个在 catch子句中说明的异常，那么

1）程序跳过 try语句块的剩余代码。2）执行 catch中的处理器代码。
