这个“某一段代码“就是我们通常所说的“回调函数”（callback）。

 

2、那么如何自己来实现一个这样的功能呢？

 很自然的可以想到，既然要对一个特定的事件做出响应，我们需要知道的事情是

  • “特定的事情发生了没有？”。
  • “我该如何响应？”

可以很容易想到类似下面的一段伪代码：

        event.addListener();
        if (event_happened) {
            run_some_codes();
        }
